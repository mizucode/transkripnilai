Petunjuk Mencetak Transkrip Nilai

  1. Gunakan browser **Chrome** atau **Firefox** untuk hasil terbaik.
  2. Tekan **Ctrl+P** (atau Cmd+P di Mac) atau klik tombol **"Cetak Transkrip"** di atas untuk membuka dialog print.
  3. Set **Destinasi** ke printer Anda.
  4. Set **Layout** ke "Portrait".
  5. Set **Ukuran Kertas** ke "A4".
  6. Set **Margin**:
    • Disarankan: "Default" atau "Minimum".
    • Hindari "None" karena akan menghilangkan margin yang sudah diatur di dokumen.
  7. Set **Skala** ke "Default" atau "100%".
  8. Pastikan opsi **"Grafis latar belakang"** (Background graphics) **DICENTANG**. Jika tidak, warna latar belakang header tabel mungkin tidak tercetak.
