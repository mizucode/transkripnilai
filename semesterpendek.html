Petunjuk Mencetak Dokumen

  1. Gunakan browser Chrome atau Firefox untuk hasil terbaik.
  2. Tekan CTRL + P (atau Cmd+P di Mac) atau klik tombol Cetak Dokumen di atas untuk membuka tampilan print.
  3. Set Destinasi sesuai kebutuhan Anda.
  4. Simpan ke PDF untuk download sebagai file PDF.
  5. Set Layout ke "Portrait".
  6. Set Ukuran Kertas ke "A4".
  7. Set Margin:
    • Disarankan: "Default" atau "Minimum".
    • Hindari "None" karena akan menghilangkan margin yang sudah diatur di dokumen.
  8. Set Skala ke "Default" atau "100%".
  9. Pastikan opsi Grafis Latar Belakang (Background graphics) Dicentang. Jika tidak, warna latar belakang header tabel mungkin tidak tercetak.
